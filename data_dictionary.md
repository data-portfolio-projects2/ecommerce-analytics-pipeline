
| Column Name | Data Type | Definition |
| :--- | :--- | :--- |
| **order_id** | object | Unique identifier for each customer order / transaction. |
| **order_item_id** | object | Unique identifier for each product item within an order. |
| **customer_id** | object | Unique identifier for each customer. |
| **product_id** | object | Unique identifier for each product. |
| **campaign_id** | object | Unique identifier for the marketing campaign associated with the order. |
| **channel_id** | object | Identifier for the acquisition channel (e.g., Paid Search, Social, Email). |
| **platform_id** | object | Identifier for the advertising platform (e.g., Google Ads, Facebook Ads). |
| **date** | object | Date the transaction or campaign activity occurred. Used for time-series analysis. |
| **customer_name** | object | Full name of the customer. |
| **email** | object | Customer email address. |
| **customer_type** | object | Classification of customer (e.g., New, Returning). |
| **country** | object | Customer country location (e.g., United States). |
| **product_name** | object | Name of the product purchased. |
| **category** | object | Product category (used as product line for performance analysis). |
| **item_price** | float64 | Selling price of a single unit of the product. |
| **quantity** | int64 | Number of units purchased in the order item. |
| **channel_name** | object | Name of the acquisition channel (e.g., Paid Search, Social, Email). |
| **platform_name** | object | Name of the advertising platform (e.g., Google Ads, Facebook Ads, Instagram Ads, Mailchimp). |
| **spend** | float64 | Advertising cost incurred for the campaign/platform on that day. |
| **clicks** | int64 | Number of ad clicks generated by the campaign. |
| **orders** | int64 | Number of orders attributed to the campaign/day. |
| **revenue** | float64 | Total revenue generated from the transaction or campaign activity. |
